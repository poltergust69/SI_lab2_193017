Објаснување на напишаните unit tests:
Кај MultipleCondition тестовите ги составуваме со цел да ја истестираме секоја можна варијанта на if условите.
А, кај EveryBranch критериумот, го тестираме секое можно гранење во кодот.
Во мојот проект ќе најдете слики од Excel каде што ова го правам за секој од критериумите.
